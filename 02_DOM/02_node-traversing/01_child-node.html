01. 자식 노트 탐색

노드 탐색

요소 노드 취득 후 취득 요소 기점으로 dom 트리의 노드를 옮겨 다니며 부모, 형제, 자식 노드 등을 탐색해야 할 때가 있다.. DOM 트리 상의 노드를 탐색할 수있도록 node, Element 는 트리 탐색 프로퍼팅를 제공한다..

자식 노드 참색

  1. node.childNodes : 자식 노드를 탐색하여 nodelist 에 담아 변환
  2. node.firstChild : 첫 번째 자식 노드
  3. node.lastChild : 마지막 자식 노드
  1. element.children : 자식 노드 중 요소 노드만 탐색하여 htmlCollection 에 담아 반환
  2. element.firstElementChild : 첫 번째 자식 요소 노드 반환
  3. element.lastElementChild : 마지막 자식 요소 노드 반환
